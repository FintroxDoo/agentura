# Contributing

Agentura is **source-available** software (see [LICENSE.md](LICENSE.md), FSL-1.1-Apache-2.0):
you are welcome to read the code, run it, and modify it for your own use.

## Pull requests

We are **not accepting pull requests** at this time. The project is developed
by a single team with a tight roadmap, and every external change carries a
review and licensing burden we currently can't take on. PRs will be closed
without review — please don't take it personally.

## Issues

**Issues are very welcome.** Bug reports, questions and feature requests all
help the project. For bug reports please include:

- your OS and Node.js version (`node --version`),
- which engine you used (Claude API / Claude Code subscription / Kimi / mock),
- the relevant lines from the server console output,
- steps to reproduce.

## Security

If you find a security issue (especially anything that lets a workspace agent
escape its sandbox or reach the host beyond the documented `run_command`
behavior), please **do not open a public issue** — email stefan@fintrox.rs.
