Dobrodošli u Agent Harness

Unesi API ključeve pre pokretanja tima

Harness koristi ove ključeve da bi agenti (programeri, team lead, QA) mogli da pozivaju modele. Dovoljno je uneti bar jedan — možeš uneti i oba i mešati motore po ulozi.

Ključ dobijaš na console.anthropic.com → sekcija API keys → „Create Key". Troši Anthropic kredite (pay-per-token).
Ključ dobijaš na platform.moonshot.ai (Moonshot / Kimi) uz Kimi For Coding pretplatu.
Restartujem server…
Ključevi se čuvaju lokalno na tvom računaru i koriste se samo za pozive modelima.
