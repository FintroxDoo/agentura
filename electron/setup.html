Enter your API keys before starting the team

Agentura uses these keys so the agents (programmers, team lead, QA) can call the models. Entering at least one is enough — you can also enter both and mix engines per role.

Get your key at console.anthropic.com → API keys section → "Create Key". Uses Anthropic credits (pay-per-token).
Get your key at platform.moonshot.ai (Moonshot / Kimi) with a Kimi For Coding subscription.
For email notifications (run finished, limit hit, stuck tasks). Get it at resend.com → API Keys. Without a verified domain, Resend only delivers to the account owner's email.
